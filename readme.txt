Coinbase \"Buy Widget\" for WordPress

== Description ==
Quick integration of the Coinbase \"Buy Widget\" for WordPress. Useful for donations or low volume, manually processed purchases.

== Frequently Asked Questions ==

= Why does the modal popup show an alert at the top of the page stating \"The widget code you provided is invalid\"? =
Your Buy Widget code from Coinbase is missing or invalid. Configure the code from Settings -> Coinbase Buy Widget.

See [Setting up the Buy Widget](https://developers.coinbase.com/docs/buy-widget#setting-up-the-buy-widget) for details
about acquiring a code.
