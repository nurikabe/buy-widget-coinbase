Coinbase "Buy Widget" for WordPress

== Description ==
Quick integration of the Coinbase "Buy Widget" for WordPress. Useful for donations or low-volume, manually processed purchases.

== Frequently Asked Questions ==

= Why does the modal popup show an alert at the top of the page stating "The widget code you provided is invalid"? =
Your Buy Widget code from Coinbase is missing or invalid. Configure the code from Settings -> Buy Widget.

= Where can I get an authorization code? =
Create new widget codes via the Settings -> API section of your Coinbase account [here](https://www.coinbase.com/buy_widgets/new).
